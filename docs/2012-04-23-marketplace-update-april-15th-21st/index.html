Marketplace update - April 15th-21st

This is a new weekly blog post about the status of the Mozilla marketplace, focusing on changes for the marketplace. For news about addons specifically, see the Addons blog.

Stats

These stats include addons.mozilla.org until we start breaking out the Marketplace components separately on Bugzilla.

  • Total bugs open: 1,363
  • Total bugs opened last week: 122
  • Total bugs closed last week: 67

Changes

  • Lots and lots of front and back end end changes as we push to the next milestone.
  • User groups and permissions were updated, you might have lost yours.
  • App reviews added.
  • Large icon support.
  • Currency selection and selection for pre-auth payments.

  • Meeting notes from last meeting.
