Arecibo open source server

Is ticking away just nicely on github. You can now happily consider that a stable stand alone version you can just take off the github shelf and install. At some point we'll start doing formal releases, but for the moment, its kind of nice not too.

In the last couple of weeks we've added in:

  • Users and security (and some doc's around that)
  • RSS feeds
  • Errors have a concept of being read
  • Some actual CSS to make it pretty

So just filtering and notifications to go. I have to say the filtering in Arecibo is a bit whacky "it seemed like a good idea at the time" so we'll have to work on that.
